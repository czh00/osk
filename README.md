# OSK (On-Screen Keyboard) 螢幕小鍵盤 ⌨️

## 專案介紹
這是一個專為 Windows 打造的**羽量級、全功能、動態可視化**虛擬螢幕鍵盤。
與內建的工具不同，我們的 OSK 針對開發者、實況主及觸控使用者做了深度優化。不僅能在畫面上帶來乾淨且一致的四角落排版，更具備實用的**動態視覺提示**、**修飾鍵雙向實時同步**以及**系統多媒體控制**等進階功能。

## 🌟 核心特色
* **雙模式版面無縫切換**
  * **100% 全尺寸版面**：包含右上角新增的多媒體控制鍵（靜音、大聲、小聲）、完整 Numpad 九宮格以及方向鍵，適合辦公與全方位輸入。
  * **60% 精簡版面**：緊湊的配置，節省螢幕空間，並透過強大的 `Fn` 與實體/虛擬組合鍵完成所有 100% 鍵盤的功能。
* **物理與虛擬無縫雙向同步**
  * 無論您是點擊螢幕上的虛擬按鍵，還是按下桌上的實體鍵盤，OSK 的按鍵都會即時亮起對應的顏色。
* **智慧「四角落」提示佈局（支援動態與靜態模式）**
  * **左上角**：預設英文字元
  * **右上角 (深藍色)**：搭配 `Shift` 輸出的符號 / 大寫
  * **左下角 (綠色)**：搭配 `Fn` 輸出的系統指令與 F1~F12 功能
  * **右下角 (橘色)**：搭配系統輸入法的中文注音符號
  * *可自訂動態模式*：在按下 Shift 或 Fn 時，鍵盤字元會「動態切換」顯示即將輸出的結果，帶來最高的使用直覺！
* **完美區分左右修飾鍵**
  * 完美支援左、右 Shift / Ctrl / Alt 的獨立識別，虛擬按鍵送出的訊號也能準確區分鍵值 (例如 `0xA0` 與 `0xA1`)，對部分遊戲或快捷鍵軟體極度友善！
* **自動日夜景主題適應**
  * 取讀 Windows 系統的個人化設定，在您切換系統深淺色主題時，OSK 會跟隨變更為低調舒適的「深色模式」或是明亮的「淺色模式」。您也能夠透過視窗上方的按鈕手動強制切換。

## 🚀 進階按鍵功能講解

### 1. `Mode` 鍵 (切換輸入法 / 預覽)
* **短按**：一鍵切換 Windows 的中/英輸入狀態（同步抓取系統 IME 狀態，右下角會有指示器提示目前是 `ㄅ` 還是 `En`）。
* **長按**：在不切換實際輸入法的情況下，長按 `Mode` 可預覽不同語言的鍵盤版面狀態。

### 2. `Fn` 鍵 (擴展功能)
按下 `Fn` 後，60% 精簡版面的按鍵將轉換為隱藏功能（以綠色字體提示）：
* `數字鍵 1~0, -, =` ➜ 轉換為 `F1` ~ `F12`
* `WASD` 附近 ➜ 轉換為 `Home`, `End`, `PgUp`, `PgDn`, `方向鍵` 等。
* `Tab` ➜ 控制台安全性選單 (包含鎖定、切換使用者、工作管理員等快速通道)
* `Backspace` ➜ `Delete`

### 3. `Num` 鍵 (獨立淺藍色標示)
完全修復的 Windows NumLock 切換功能。在 100% 版面下，當 NumLock 啟動時按鍵會呈現淺藍色高亮；當關閉時，Numpad 區塊也會自動切換為對應的「上下左右」圖示。

### 4. `Shift` 鍵 (臨時英文模式 / 符號切換)
* 當處於 **中文(注音)模式** 時，點擊 `Shift` 可以快速進入 Windows 預設的 **「臨時英文模式」**。此時鍵盤上的按鍵會切換回英文字母與普通符號。
* Numpad 與主要英數區的 Shift 狀態獨立結算，以防誤觸 Windows 內建的反向 NumLock 切換機制。

## 📥 安裝與執行
1. 請確保您的系統環境已安裝 `.NET 8.0 Desktop Runtime`。
2. 下載或 Clone 本專案。
3. 使用 `dotnet build` 或 Visual Studio 建置。
4. 執行生成的 `OSK.exe`。
5. 點擊視窗左上角的「圖釘(📍/📌)」按鈕，決定視窗是否永遠置頂且點擊穿透，📍表示常駐頂端，📌會根據是否輸入文字自動顯示或隱藏。
